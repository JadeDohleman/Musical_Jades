Musical Jades is a Python-based audio engine with a graphical user interface.

## Dependencies

- Python 3.11+
- NumPy
- SciPy
- Pygame

## Usage

```bash
python src/gui.py
```

## Configuration

```json
{
    "sample_rate": 44100,
    "buffer_size": 512,
    "volume": 0.5,
    "polyphony": 16
}
```
